Day One Path

flowchart TB
    A[New user] --> H[/gsd-help/]
    H --> NP[/gsd-new-project/]
    H --> NW[/gsd-new-workspace/]

    NP --> D[/gsd-discuss-phase/]
    NP -. optional .-> SP[/gsd-spec-phase/]
    SP -. clarifies WHAT .-> D

    D --> PL[/gsd-plan-phase/]
    PL --> EX[/gsd-execute-phase/]
    EX --> VW[/gsd-verify-work/]
    VW --> SH[/gsd-ship/]
    SH --> CM[/gsd-complete-milestone/]

    VW --> NX[/gsd-next/]
    NX --> PL

    subgraph Learn[Learn the surfaces]
      D --> UI[/gsd-ui-phase/]
      PL --> RE[/gsd-research-phase/]
      PL --> PR[/gsd-plan-review-convergence/]
      EX --> SR[/gsd-session-report/]
    end

      H --> DO[/gsd-do/]
      H --> NT[/gsd-note/]
      H --> EXPL[/gsd-explore/]
    end

When Things Go Sideways

flowchart TB
    A[Something broke] --> P[/gsd-progress/]
    A --> F[/gsd-forensics/]
    A --> H[/gsd-health/]
    A --> D[/gsd-debug/]

    P --> Q{What is wrong?}
    Q -->|workflow state| F
    Q -->|planning dir health| H
    Q -->|bug or failing behavior| D
    Q -->|you need to stop and hand off| PW[/gsd-pause-work/]

    PW --> RWK[/gsd-resume-work/]
    RWK --> P

    F --> UND[/gsd-undo/]
    F --> REP[/gsd-reapply-patches/]
    F --> CL[/gsd-cleanup/]
    F --> IMP[/gsd-import/]

    H --> ST[/gsd-stats/]
    H --> CT[/gsd-check-todos/]
    H --> AT[/gsd-add-todo/]
    H --> IDOC[/gsd-ingest-docs/]

    D --> ADT[/gsd-add-tests/]
    D --> UAT[/gsd-audit-uat/]
    D --> SEC[/gsd-secure-phase/]

    UND --> P
    REP --> P
    CL --> P
    IMP --> P
    ADT --> P
    UAT --> P
    SEC --> P

Phase & Milestone Management

flowchart TB
    A[Need to change roadmap scope] --> Q{What kind of change?}

    Q -->|append new future work| ADD[/gsd-add-phase/]
    Q -->|urgent work between phases| INS[/gsd-insert-phase N/]
    Q -->|remove unstarted future work| REM[/gsd-remove-phase N/]
    Q -->|inspect assumed approach| LPA[/gsd-list-phase-assumptions N/]
    Q -->|dependency concern| DEP[/gsd-analyze-dependencies/]
    Q -->|gap after milestone audit| GAP[/gsd-plan-milestone-gaps/]

    ADD --> RM[ROADMAP.md updated]
    INS --> DEC[Decimal phase created<br/>example: 3.1]
    REM --> REN[Future phases renumbered]
    LPA --> ASSUMP[Assumptions surfaced before planning]
    DEP --> ORDER[Phase ordering/dependency report]
    GAP --> GPLAN[Gap-fix planning work]

    RM --> NEXT[/gsd-next/]
    DEC --> NEXT
    REN --> NEXT
    ASSUMP --> DISC[/gsd-discuss-phase/]
    ORDER --> PLAN[/gsd-plan-phase/]
    GPLAN --> PLAN

    classDef key fill:#fef3c7,stroke:#d97706,stroke-width:2px,color:#111827;
    class INS key;
